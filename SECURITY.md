# 安全策略（SECURITY）

## 支持的版本

`IMBotCore` 目前以 `main` 分支为主线迭代；安全修复会优先合入 `main`，并在需要时补充发布说明（见 `CHANGELOG.md`）。

## 报告安全漏洞

请不要在公开 Issue、讨论区或 PR 中披露潜在安全问题。

推荐使用 GitHub 的 **Security Advisories** 私密通道进行报告：

- 进入仓库页面 → `Security` → `Report a vulnerability`

报告内容建议包含：

- 漏洞影响范围与可能的攻击路径
- 复现步骤（最小可复现）
- 受影响的版本/提交范围（如果已知）
- 任何可用的临时缓解方案

## 响应与披露

- 我们会尽快确认收到并进行初步评估
- 修复完成后，会在合适的时间窗口内进行披露与发布说明
